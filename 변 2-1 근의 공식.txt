주요 내용 :
실수형 변수 a, b, c를 정의하고 입력받은 후 실수형 변수 dis에 대한 식을 만든다.(dis = b^2 - 4.0*a*c)
근을 구하기 위해 입력받은 값에 대한 조건들을 따진다. 
우선 (1) a=0, (2) a != 0이고 b = 0인 경우, (3) a != 0이고 b != 0인 경우로 나눈다.
(1)의 경우 dis 값에 상관없이 -c / b라는 실근 하나를 가진다.(단 b=0일때 근은 c이다.)
(2)의 경우 -c/a에 루트를 씌운 값이 발생한다. -c/a > 0이면 실근 2개 -c/a < 0 허근2개를 가진다.
(3)의 경우 dis =0 이면 중근 한개, dis >0이면 실근 2개, dis < 0 이면 허근 2개를 가진다.

*주의할 점 : 허근일 때는 루트안에 - 값이 존재함으로 컴퓨터에서 구현이 안된다.
그래서 루트안에 - 를 한번 더 씌우주고 출력시 i값을 붙여서 출력이 되게 구현한다.
----------------------------------------------------------------*/
#include <stdio.h>
#include <math.h>
#include <stdlib.h>


int main(void)
{
	double a, b, c, dis;  //dis 는 판별식이다.

	printf("계수 a를 입력하시오: ");
	scanf("%lf", &a);
	printf("계수 b를 입력하시오: ");
	scanf("%lf", &b);
	printf("계수 c를 입력하시오: ");
	scanf("%lf", &c);

	dis = b*b - 4.0*a*c;

	if (a == 0 && b != 0)                                           //(1) a=0이면  일차방정식. (b != 0인 경우) 
		printf("방정식의 근은 %f하나입니다.", -c / b);

	else if (a == 0 && b == 0)                                      // a=0이면  일차방정식. (b = 0인 경우)
		printf("방정식의 근은 %f하나입니다.", c);
	
	else if ((b == 0) && ((-c / a) >= 0))                      // b=0 이면 ax2+c=0으로 -c/a의 값에 따라서 실근 또는 허근을 가진다.
 	{
		printf("방정식의 첫번째 근은 %f입니다(실근).\n", (sqrt(-c / a)));
		printf("방정식의 두번째 근은 %f입니다(실근).", (-sqrt(-c / a)));
	}

	else if ((b == 0) && ((-c / a) < 0))
	{
		printf("방정식의 첫번째 근은 %fi입니다(허근).\n", (sqrt(c / a)));
		printf("방정식의 두번째 근은 %fi입니다(허근).", (-sqrt(c / a)));
	}
	
	else if (dis == 0)                                         //dis=0이면 중근을 가진다. 
		printf("방정식의 근은 %f하나입니다.", -b / (2.0*a*c));

	else if (dis > 0)

	{
		printf("방정식의 첫번째 근은 %f입니다(실근).\n", (-b + sqrt(dis)) / ((2.0)*a));
		printf("방정식의 두번째 근은 %f입니다(실근).", (-b - sqrt(dis)) / ((2.0)*a));
	}


	else if (dis < 0)      // 정수부분과 허수부분으로 나눠서 계산해야한다. 
	{
		printf("방정식의 첫번째 근은 %f+%fi입니다(허근).\n", -b / ((2.0)*a), (sqrt(-dis)) / ((2.0)*a));
		printf("방정식의 두번째 근은 %f-%fi입니다(허근).", -b / ((2.0)*a), (sqrt(-dis)) / ((2.0)*a));
	}



	return 0;
}
